# SafeZone - 범죄예방 길안내 서비스

> **AI 기반 안전 경로 추천 모바일 어플리케이션**  
> 공공데이터(CCTV, 가로등, 범죄율 등)를 활용하여 사용자의 이동 경로를 분석하고  
> 위험도가 낮은 **가장 안전한 길**을 안내합니다.

---

## 프로젝트 개요

**SafePath**는 사용자의 위치 정보를 기반으로  
CCTV, 가로등, 범죄 발생률 등 **공공데이터**를 분석하여  
AI가 안전 점수를 산출하고 **“가장 안전한 경로”**를 추천하는 서비스입니다.

- 사용자 인증 및 JWT 기반 보안 처리
- 안전 점수 산정 AI 모델 적용
- 커뮤니티를 통한 사용자 간 정보 공유
- 실시간 위치 기반 안전 알림 기능

---

## 주요 기능

### 회원 관리 (Auth / User)
- 회원가입 / 로그인 / 로그아웃
- 아이디 찾기 및 비밀번호 재설정
- 내 정보 조회 및 수정
- 회원 탈퇴

### 데이터 기반 서비스
- CCTV, 가로등, 범죄율 등의 공공데이터를 활용한 지역 안전 점수화
- AI 기반 안전 경로 추천 기능
- 실시간 위험 알림 시스템

### 커뮤니티
- 게시글
- 댓글 / 대댓글
